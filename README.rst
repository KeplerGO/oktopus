🐙 Oktopus
==========
|ci-badge|

.. |ci-badge| image:: https://travis-ci.org/mirca/oktopus.svg?branch=master

A soft-bodied, eight-armed package for beautiful inference.

To install ``oktopus``, run::

    pip install oktopus

or::

    git clone https://github.com/mirca/oktopus.git
    cd oktopus
    pip install -e .

And you're all set!
